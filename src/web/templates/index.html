How it works

Set up Shopkeep in under two minutes — no technical knowledge required.

01

Add Shopkeep to Discord

Click Add to Discord and select the server where you want order notifications.

02

Get your setup link

Shopkeep sends you a private DM with a link to connect your Etsy shop.

03

Authorize with Etsy

Follow the link and grant Shopkeep read access to your Etsy orders in one click.

04

Choose a channel

Run /setchannel in Discord to pick where notifications appear. That’s it.

FAQ

Common questions about Shopkeep.

Is Shopkeep really free?
Yes, completely free. No plans, no limits, no credit card required — ever. Shopkeep is a passion project built for Etsy sellers.
How quickly does it detect new orders?
Shopkeep checks your Etsy shop every 60 seconds, so you'll see a notification within a minute of a new order being placed.
Can I use it with multiple Etsy shops?
Each Discord server connects to one Etsy shop. If you have multiple shops, just add Shopkeep to multiple Discord servers — one per shop.
What Etsy permissions does it request?
Read-only access to your transactions, listings, and shop profile. Shopkeep can never modify your Etsy shop, post listings, or make any changes to your account.
Who can configure the bot in my server?
Setup requires server ownership (the initial connect link is sent to the owner). After that, anyone with the Manage Channels permission can run /setchannel to change where notifications go.
What if my Etsy token expires?
Shopkeep handles token refresh automatically in the background. If it ever can't refresh, it'll post a message in your notifications channel with a re-auth link so you never miss an order.
