Changelog

What’s new

A running log of improvements and new features.

v0.4.0 April 2025 Latest
new Shipping presets — save and reuse carrier + package configurations with /preset add, /preset list, and /preset remove.
new Light and dark mode toggle on the website, with system preference detection.
fix Improved token refresh reliability — Shopkeep now proactively refreshes tokens 60 seconds before expiry.
v0.3.0 March 2025
new Revenue summaries — /revenue shows totals for today, this week, or this month.
new Shipping reminders — automatic daily alerts for open orders approaching their ship-by date, color-coded by urgency.
new Order variations now shown inline on notifications instead of hidden in spoiler text.
fix Shipping reminder urgency colors (red/yellow/green) now correctly reflect days remaining.
v0.2.0 February 2025
new /orders — list recent orders filtered by status and date range.
new /shop — display connected shop info and current status.
new /listings — browse active Etsy listings from Discord.
new Multi-server support — each Discord server maintains its own isolated shop connection and settings.
v0.1.0 January 2025
new Initial release — real-time Etsy order notifications posted to a Discord channel.
new Secure OAuth 2.0 + PKCE setup flow — connect your Etsy shop in under two minutes.
new /setchannel and /status slash commands.
